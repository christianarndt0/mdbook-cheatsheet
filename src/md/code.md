# Code Snippets

## Markdown

````markdown
Inline code has `back-ticks around` it.

```python
s = "hello"
print(s)
```

```rust
let s = "jello";
println!(s);
```
````

## renders as

Inline code has `back-ticks around` it.

```python
s = "Jello"
print(s)
```

```rust
let s = "jello";
println!("{}", s);
```
